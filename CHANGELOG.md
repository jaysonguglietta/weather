# Changelog

## 1.7.0

- Added Local Weather Trivia cards for daylight duration, temperature swing, humidity feel, wind direction, and rain probability.

- Added animated weather scenes with falling rain, snow, drifting fog, storm flashes, moving clouds, and wind-blown leaves.
- Kept the scene compatible with reduced-motion preferences.

## 1.5.0

- Added dog park recommendations with sunrise, sunset, and humidity context.

## 1.4.0

- Added daily umbrella, outfit, comfort, and weather gear recommendations.

- Added a Weather News section with National Weather Service alerts and fallback weather-resource links.

- Added an explicit City/ZIP selector and dedicated US ZIP code lookup.

## 1.1.0

## 1.0.0

- Initial WeatherBoard web application.
- Added current conditions, hourly chart, seven-day forecast, saved locations, unit switching, geolocation, static PWA assets, and documentation.
